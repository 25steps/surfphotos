surfphotos - фотохостинг, при регистрации в котором можно загрузить фотографии и любой пользователь сможет просмотреть их и скачать оригинал.
Приложение собирает статистику просмотров и скачиваний, при этом можно отсортировать фотографии по популярности.

Доступ осуществляется через 3 домена:
Веб интерфейс доступен на домене surfphotos.com;
REST сервисы - на домене api.surfphotos.com;
SOAP веб сервисы - на домене soap.surfphotos.com;

Обзор функций приложения surfphotos.com:
Отображение пользовательского профиля и его фотографий;
AJAX загрузка следующей порции фотографий;
Регистрация через Facebook;
Регистрация через Google+;
Транслитерация имени и фамилии пользователя;
Автоматическая генерация UID пользователя на основании его имени и фамилии;
Скачивание аватарки с социальной сети;
Загрузка новой аватарки;
Редактирование персональных данных;
Отображение фотографий по популярности фотографий;
Отображение фотографий по популярности автора;
Просмотр (с увеличением счетчика просмотров) большой фотографии;
Скачивание (с увеличением счетчика скачиваний) оригинальной фотографии;
Загрузка новых фотографий;
Хранение трех версий фотографии: маленькой - для отображения в UI, большой - для предпросмотра, оригинальной - для скачивания;
User Friendly url и сообщения об ошибках;
Ежедневное обновление рейтинга пользователя по формуле: rating = views * 1 + downloads * 100;

Используемый стек технологий:
БД - PostgreSQL 12;
Backend - Java EE 7;
Frontend - JSP, JS, CSS;
Серверная часть - для приложения GlassFish 4.1.1, для статики(картинки и тд) Nginx 1.12.0;
Для тестирования REST сервисов используется swagger api;
